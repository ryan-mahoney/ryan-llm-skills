# Table & Row Design Guide (v1.1)

**Core rule:** Tables exist to find, compare, or act.

## 1. Structure
- Horizontal separators by default
- No vertical gridlines unless needed
- Consistent density

## 2. Typography
- 13–14px body
- 12–13px headers
- Tabular numerals
- No ALL CAPS

## 3. Alignment
- Text left
- Numbers right
- Headers match cells
- Actions in fixed zone

## 4. Headers
- Always visible
- Terse labels
- Clear sort indicators

## 5. Cell Content
- Truncate long text
- Use subtext to reduce columns
- Status = color + text
- Primary identifier is link

## 6. Row Interaction
- Full-row hover
- Clickable rows only when safe
- Selection via first column

## 7. Actions
- 1–2 inline
- 3+ in menu
- Bulk via contextual toolbar

## 8. Search & Filter
- Above table
- Filters visible and removable

## 9. Responsive
- Desktop: table
- Tablet: hide secondary columns
- Mobile: card/list view

## 10. Accessibility
- Semantic table markup
- aria-sort
- Keyboard operable

## 11. Final Test
1. Task clear?
2. Any removable column?
3. Comparison effortless?
